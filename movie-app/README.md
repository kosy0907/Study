# Movie App

## Skill
* FrontEnd
    * React.js

## 사용 API
* TMDB movie api

## Page
* Login
    * Firebase 기반 Email & PW 로그인
* Home
    * 최근 인기 영화 Title, Rating  
* movie detail
    * Home에서 선택한 영화의 상세 info

## 기능
* 최근 인기 있는 영화 확인
* 영화 정보 확인
    * Title
    * Release Date
    * Language
    * Run Time
    * Actor info
